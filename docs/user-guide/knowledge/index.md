## Overview

Knowledge is the platform's internal reference library for reusable guidance, decisions, and operating notes. It gives teams and AI coworkers a shared place to find durable context instead of rediscovering the same answer in every workflow.

## Key Concepts

- **Articles** — Focused reference pages for procedures, decisions, troubleshooting notes, or reusable explanations.
- **Operational context** — Practical guidance that helps staff and coworkers make consistent decisions.
- **Linked work** — Knowledge can reference backlog items, products, customers, architecture decisions, or compliance evidence.
- **Review cadence** — Important articles should be reviewed when related workflows or policies change.

## What You Can Do

- Browse and search internal guidance
- Create articles for repeatable procedures or decisions
- Link knowledge back to the workflow it supports
- Keep AI coworker context aligned with current operating practice
- Retire or update stale articles when the product changes
