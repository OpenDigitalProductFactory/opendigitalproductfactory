## Workflow

1. Create or review the bank account record.
2. Import transactions when new statement activity arrives.
3. Use rules to automate categorisation where the pattern is stable.
4. Reconcile unmatched transactions before closing the period.

## What To Watch

- imported transactions that remain unmatched
- rules that over-apply or classify to the wrong ledger intent
- reconciliation steps blocked by missing bills, invoices, or supplier/customer setup
