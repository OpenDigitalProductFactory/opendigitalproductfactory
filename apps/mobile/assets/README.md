# App Assets

Required before app store submission:

- `icon.png` — 1024x1024 app icon (PNG, no transparency for iOS)
- `adaptive-icon.png` — 1024x1024 Android adaptive icon foreground
- `splash.png` — 1284x2778 splash screen image

Design should match the DPF platform branding (dark theme, indigo accent #818cf8).
